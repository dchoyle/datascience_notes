# datascience_notes
Repository for the series of 'Data Science Notes' blogposts at www.hoyleanalytics.org

For these Data Science notes you will need the following:
1.	A Python installation, including the packages
    *	pandas (>=2.0.3)
  	*	NumPy (>=1.24.3)
  	*	matplotlib (>=3.7.2)
  	*	statsmodels (>=0.14.0)
   *	SciPy (>= 1.13.1)

2.	The ability to view and run Jupyter notebooks.
These requirements can be met by installing a full Anaconda distribution (from https://www.anaconda.com/download)
